# Software Engineer Portfolio

This project is now a Next.js portfolio website using the App Router.

## Project Structure

- `app/layout.js`: Root layout and metadata
- `app/page.js`: Main portfolio page
- `app/globals.css`: Global styles and glassmorphism design
- `package.json`: Dependencies and scripts

## Get Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Then open:

`http://localhost:3000`

## Build For Production

```bash
npm run build
npm run start
```

## Customize Before Publishing

Replace the placeholder content in `app/page.js` with your real:

- Name
- Job title
- Bio
- Skills
- Project details
- Email
- GitHub link
- LinkedIn link
- Resume link
